Classes

Monday April 1, 2019 - What Am I Eating?

This food-based class will help kids discover what's in the food they are eating every day. We will do experiments to extract and analyze iron, starch, sugar, and fat from various foods we eat, as well as experiment with yeast and fermentation.

Wednesday April 3, 2019 - Can You Engineer It?

This class is all about engineering challenges. We will be building spaghetti towers, newspaper bridges and clay boats, and protecting a Pringle from a falling object using only paper and tape.

Thursday April 4, 2019 - Science Is Magic!

This class teaches magic tricks based on physics and chemistry! We will be making invisible ink, levitating orbs, and learning about the invisible forces all around us.

The cost for each class is $20/child.
